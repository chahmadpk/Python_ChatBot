# Python Chatbot

A beginner-friendly Python chatbot with a modern graphical user interface.
This application simulates basic conversation using rule-based logic and provides a clean and interactive chat experience.

---

## Features

* Interactive chat window
* Send button + Enter key support
* Modern dark theme UI
* User and Bot message differentiation
* Rule-based intelligent responses
* Custom chatbot identity

---

## Technologies Used

* Python
* Tkinter (GUI)
* ScrolledText
* Random module

## How to Run

1. Clone the repository:

```
git clone https://github.com/your-username/chatbot-gui.git
```

2. Run the program:

```
python chatbot.py
```
